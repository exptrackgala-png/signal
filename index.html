⚡ Zerodha Kite API Setup
1. Visit developers.kite.trade
2. Create app → App type: Connect
3. Redirect URL: http://localhost
4. Copy API Key & Secret
5. Login URL: kite.zerodha.com/connect/login?api_key=XXX
6. Grab request_token from redirect URL
7. Exchange for access_token (refreshed daily)
Get from: kite.trade redirect URL after login
⚠ Browser CORS blocks direct Kite API calls. Orders route through a local Node.js proxy (port 3001). Without it, orders are simulated and logged only. See README for proxy setup.

⚡ Connect Zerodha Kite
1. developers.kite.trade → My Apps → Create New App (type: Connect)
2. Redirect URL: http://localhost · Copy API Key from app page
3. Open: https://kite.zerodha.com/connect/login?api_key=YOUR_KEY
4. After login, copy request_token from the redirect URL
5. Exchange it for access_token (valid for 1 day) — use Kite Python SDK or Postman
6. Paste access_token below. Re-do daily after market open.
⚠ Browser CORS blocks direct Kite API. Run the included Node.js proxy (proxy.js) on localhost:3001 for live order execution. Without it orders are simulated.
